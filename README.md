# Embedded Workbench

**Embedded Workbench** is a learning and working environment for embedded programming, offering a variety of tools and services to help control and interact with hardware. The project is organized into three main areas: Hardware Analysis, Hardware Programming, and Hardware Access. Each section will be described below, along with its corresponding workflow.

---

## Services

### Hardware analysis

**gpio-uart-i2c-spi-logic-analyzer**

**usb-uart-jtag-swd-monitor**

**software-defined-radio-monitor**

**network-monitor**

### Hardeware programming

**platformio**

**pic**

**raspberrypi**

**spartan_6**

### Hardware access

**Linux**

**Android**

**Windows**

**Mac Os**

**Ios**

### Set up the environment

**Clone the repository**

```
git clone https://github.com/Rhaytou/embedded-workbench.git
```

**Change directory**

```
cd ./embedded-workbench
```

### General workflow 

**Note**:This is a general workflow, it will up all services you must see the 'General informations' section.

**Start all services**

```
make up
```

**Down all services**

```
make down
```

**Clean all docker**

```
make docker_clean_all
```
